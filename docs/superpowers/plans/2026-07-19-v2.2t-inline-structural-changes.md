# v2.2T Inline Executive Milestone Changes Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the form-led Executive structural-change workflow with contextual card/cell actions and make monthly-update freshness unambiguously separate from milestone health.

**Architecture:** Keep the existing callable payload and approval/audit model unchanged. Add a small freshness presentation contract in `js/executive-governance.mjs`; both dashboard entry points consume it. Rework the existing structural-change overlay into a contextual action sheet that derives item identity, source, version, insertion position, and exact request diff from its session instead of asking a requester to provide them.

**Tech Stack:** Static HTML/CSS/ES modules, Firebase Web SDK, Firebase callable Functions, Node built-in test runner.

## Global Constraints

- RAG is the only coloured milestone-health indicator.
- Freshness text must use `Update record`; never render `Overdue`, `Not started`, `Current`, or `Update due`.
- Freshness is neutral gray-blue metadata, never a schedule-delay or RAG signal.
- `Refresh requested` starts after 31 days and `Please refresh` after 45 days; a missing record is `No update recorded`.
- Structural requests still require a reason and preserve exact `before`/`after` payloads, expected version, and current authorization.
- Admin and Executive Owner directly apply structural changes; all other approved roles submit for approval.
- Released weeks remain read-only. Do not change callable contracts, approval collections, audit schema, or the documented deployment blockers.

```js
export function getExecutiveUpdateFreshness(lastUpdatedAt, now = Date.now()) {
  const updatedAt = typeof lastUpdatedAt === 'number' ? lastUpdatedAt : Date.parse(String(lastUpdatedAt || ''));
  const currentTime = typeof now === 'number' ? now : Date.parse(String(now || ''));
  if (!Number.isFinite(updatedAt)) return 'missing';
  if (!Number.isFinite(currentTime)) return 'current';
  const elapsedDays = Math.max(0, (currentTime - updatedAt) / DAY_MS);
  if (elapsedDays > 45) return 'please-refresh';
  if (elapsedDays > 31) return 'refresh-requested';
  return 'current';
}
```
